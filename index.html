应用简介 / App Overview

核心定位 / Core Positioning:
为头显沉浸式场景设计的轻量化控制中枢，通过手机远程管理 VR 头显中的全景视频与 3D 场景，实现无干扰的沉浸体验切换。
A lightweight control hub designed for immersive VR headsets, allowing users to remotely manage panoramic videos and 3D scenes via mobile phone for seamless experience switching.

核心功能 / Core Features:

  • 自然疗愈全景库：4K 超清山川/竹林/海底/冰雪等自然场景
    Nature Panorama Library: 4K ultra-HD nature scenes such as mountains, bamboo forests, oceans, and snowy landscapes
  • 极简卡通场景集：低面数抽象化 3D 空间（如沙漠、森林、冰原），降低视觉负荷
    Minimalistic 3D Cartoon Scenes: Low-poly stylized spaces (e.g., desert, forest, ice field) to reduce visual strain
  • 一键热切换：手机端点击场景缩略图，头显 0.5 秒内完成切换
    Instant Switching: Tap a scene thumbnail on your phone to switch the VR display within 0.5 seconds
  • 多设备看板：同时监控多台头显状态，独立控制各设备播放内容
    Multi-device Dashboard: Monitor and control multiple headsets individually in real-time

操作流程 / User Flow:
选择设备 → 浏览场景库 → 点击切换 → VR 头显实时响应 → 查看播放状态
Select Device → Browse Scenes → Tap to Switch → VR Responds Instantly → Monitor Playback
